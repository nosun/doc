## 二、 sysstat
- yum list sysstat
- cat cron.d/sysstat
crontab 每分钟去获取一遍操作系统的状态信息，存储在/var/log/sa目录下
- sar -q -f cpu 
- sar -p -f 
- sar -r -f 内存，磁盘
- sar -B -f 内存翻页，内存不够用了，会动用磁盘，对系统io会造成影响。
- sar -W -f swap使用情况
- sar -b -f 磁盘io分析
- sar -d -f 磁盘io分析 具体到扇区，以及磁盘读写占用cpu的情况
- sar -n DEV -f
